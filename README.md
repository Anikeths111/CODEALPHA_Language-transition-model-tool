# 🌐 Language Translation Tool
### CodeAlpha AI Internship — Task 1

A clean, fully functional Language Translation Web App 
built with Streamlit and Google Translate API.

---

## ✨ Features
- 🌍 Translate text across 25+ languages
- 🔍 Auto-detect source language
- ⇄ Swap source and target languages
- 📋 Copy translated text to clipboard
- 🔊 Text-to-Speech — listen to translation
- ⚡ Fast and responsive UI

## 🛠️ Tech Stack
- Python
- Streamlit
- deep-translator (Google Translate API)
- gTTS (Google Text-to-Speech)

## 🚀 How to Run

### 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/CodeAlpha_LanguageTranslation.git

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run the app
streamlit run "language transition tool.py"

### 4. Open in browser
http://localhost:8501
